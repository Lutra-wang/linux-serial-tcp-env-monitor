# Linux 串口数据网关项目

## 一 项目简介

本项目基于 Linux 实现一个串口数据网关系统，用于将嵌入式设备采集的数据接入 Linux 系统，并通过网络进行转发。系统能够完成从串口数据接收、协议解析到 TCP 数据发送的完整流程。

该项目主要用于嵌入式开发与 Linux 系统编程学习，同时具备简单物联网网关的功能雏形。

---

## 二 项目目的

本项目的主要目的包括

- 掌握 Linux 下串口设备的使用方法
- 理解 termios 串口配置机制
- 实现基于状态机的协议解析
- 掌握 TCP socket 编程基本流程
- 打通设备侧到网络侧的数据链路
- 提升模块化设计和工程组织能力

---

## 三 项目内容

项目主要包含以下几个核心功能模块

1 串口数据接收  
从设备文件 dev ttyACM0 读取 MCU 发送的数据

2 协议解析  
基于状态机解析自定义数据帧，支持不定长数据流

3 数据处理  
从数据帧中提取温度和湿度信息

4 网络转发  
作为 TCP 服务端，将解析后的数据发送给客户端

5 模块化设计  
将串口 协议 网络 分模块实现，结构清晰

---

## 四 实现效果

系统运行后，可以实现以下效果

- 成功接收 MCU 串口发送的数据
- 正确解析完整数据帧，避免半包和粘包问题
- 提取温湿度数据并进行处理
- 客户端连接后，可以实时接收数据

示例输出

TEMP 25.3 HUM 60.8

MCU 到 Linux 到 TCP 客户端
